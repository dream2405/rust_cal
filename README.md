# Rust Calculator
중위 표기법으로 된 식을 입력받고, 계산 결과를 출력하는 계산기

### 입력
중위 표기법으로 된 식을 입력  


#### 지원되는 연산자  
`+ = * / ( )`  
#### 지원되는 피연산자
음수 포함 실수  
음수를 표기할때 -를 붙여 써야 한다  
ex)      
`23.253 -3.0 0 0.004`  


### 출력  
ex)  
입력  
`-1.6 + 4.7`  
`( ( 1 + 2 ) )`  
`43.5 * -22.4 + ( 61.28 / ( -12.23 + 63.25 ) )`  
출력  
`결과: 3.1`  
`결과: 3`  
`결과: -973.1989023912191`

### 입력 규칙  
+ 중위 표기법으로 입력해야 한다.
+ 연산자와 피연산자 사이를 띄어쓰기 한다. (괄호도 포함)
